# YouTube Downloader Web (Fixed Version)

## How to run locally
1. Unzip the folder.
2. Open terminal and navigate to the folder.
3. Run:
   ```bash
   pip install -r requirements.txt
   python app.py
   ```
4. Visit `http://127.0.0.1:5000` in your browser.

## Add logo
Copy your `yt-logo.png` file into `static/` folder.

Then reload the page to see the logo.
